# CHONKPUMP 9000 - $CHONK9K Token Game

A production-ready viral tap-to-earn game featuring the $CHONK9K token. Built with Next.js 16, React 19, Supabase, and optimized for mobile-first PWA experience.

## Features

- **Earn $CHONK9K Tokens**: Tap to earn real $CHONK9K tokens that are saved to your profile
- **10 Evolution Levels**: Watch your CHONK evolve through stunning visual transformations
- **Power-Up System**: Unlock 2x, 3x, and 5x multipliers with special effects
- **Global Leaderboards**: Daily, weekly, and all-time rankings with real-time updates
- **Social Sharing**: Share your $CHONK9K earnings and compete with friends
- **Progressive Web App**: Install on any device, works offline, native app experience
- **Secure Authentication**: Production-grade Supabase Auth with email/password
- **Real Database**: PostgreSQL with Row Level Security for data protection
- **Mobile-Optimized**: Haptic feedback, smooth animations, 44px touch targets
